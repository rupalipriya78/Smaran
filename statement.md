Problem Statement
Students manage multiple deadlines, exams, and to-do tasks across various subjects. 
Without a unified system, tasks get forgotten, deadlines are missed, and stress increases. 
A structured platform is required to track all tasks with options to add, view, 
complete, and delete items.

Scope
Smaran allows students to track:
- Assignment deadlines  
- Exam schedules  
- Daily to-dos  

The system provides:
- UI for adding and viewing tasks  
- Python backend for logic  
- Strike-through completion  
- Deletion of unnecessary tasks  

Target Users
- College students  
- School students  
- Anyone managing academic tasks  

High-Level Features
- Add/View/Delete tasks  
- Category-wise task management  
- Built-in strike-through completion  
- Aesthetic and minimal UI
